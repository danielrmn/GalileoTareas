B. Cuando se trata de condicionales, podemos decir que el if tiene 3 formas

IF

Este se representa de manera contidianda como: "Si esto pasa".

Se utiliza en cada operacion logica

Ejemplos:

1. cuando queremos escoger que comer en el desayuno, SI hay huevos, comere un omelette.

2. Al comprar en el super, SI me sobrea dinero, comprere un voleto de loteria.

ELSE iF

Este al igual que el anterior, reprecenta una condicion pero tambien una excepcion.

Se utiliza cuando la primera condicion no se logra cumplir

Ejemolos:

1. AL comprar una bebida, SI hay sabor naranja, lo compro. DE LO CONTRARIO comprare el de uva.

2. escoger una pelicula, SI estoy aburrido, veo rapidos y furiosos. DE LO CONTRARIO vere anime

ELSE

Este es el ultimo recurso de las condicionales

Aca es cuando la condicion tiene que hacer algo por defecto tecnico.

Ejemplos:

1. Al ir un restaurante, SI todavia hay desayuno comopro, si no toca almuerzo

2. Al comprar el nuevo iphone, SI todavia hay disponible, si no hay, no lo compro
